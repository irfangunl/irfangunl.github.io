About

/ who i am

I'm a software engineer who likes shipping things that work. Real users, real edge cases, real production feedback — that's where I learn the most.

I'm currently a Software Engineering Intern at Miote, and I'm graduating soon from my IT degree. On the side I build open-source tools (VerusPanel, VerusDB, RouteZero) and smaller experiments in AI, data science, and productivity software. Most of my work lives on GitHub.

My strongest area is the PHP / Laravel / MySQL stack — that's what I reach for when I'm building something that needs to be reliable, boring in a good way, and ready for real users. Outside of that, I work in JavaScript/TypeScript (React, Node.js), Python (for ML and scripting), and I've shipped production code in every layer from the database to the UI.

I'm not a fan of over-engineering. I'd rather write 200 lines of clear code than 2,000 lines of "framework-correct" code. The point of software is to be useful, not to win architecture astronaut awards.
